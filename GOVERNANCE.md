# Governance

MathEvidence is governed as shared formal-mathematics infrastructure.

Maintainer areas:

- Core and trust model
- Semantic IR
- Domain checkers
- Backend adapters
- Agent API
- Studio
- Registry
- Foundry and benchmarks
- Security and releases

Stable protocol changes require two approvals from different areas. Domain semantics require relevant mathematical expertise. No single maintainer may unilaterally weaken replay, axiom, request-binding, or claim-strength guarantees.
